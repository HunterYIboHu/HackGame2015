About

本大赛由信息科学与工程学院团委学生会主办，兰州大学开源社区、兰州大学华为创新俱乐部承办。

本大赛对参赛者的基本网络攻防能力进行测试，游戏简单有趣，欢迎全校同学参加。

无后台、无黑幕、无危险的“三无”大赛！

Web 攻防

（1）参赛者通过参加在线渗透闯关测试完成比赛，需要游戏方向为 Web 渗透游戏。

（2）参赛者攻破游戏服务器，包括但不限于使之瘫痪、任意文件下载、Getshell。

二进制安全

（1）参赛者通过对二进制文件进行处理，使之达到免杀效果

（2）参赛者尝试破解特定程序
